## How to Run

### Option 1: Python (Recommended)
```bash
cd alvverse
python3 -m http.server 8000
```

Then open http://localhost:8000 in your browser.

### Option 2: Node.js
```bash
cd alvverse
npx http-server -p 8000
```

### Option 3: PHP
```bash
cd alvverse
php -S localhost:8000
```

## Features

- Clean, simple HTML/CSS structure
- No complex dependencies
- Responsive design
- Easy to customize
- Comic Sans MS font throughout
- Animated floating elements

## Customization

- **Colors**: Edit the color values in `css/style.css`
- **Images**: Replace images in the `images/` folder (keep the same names)
- **Content**: Edit text directly in `index.html`
- **Layout**: Modify positioning in `css/style.css`

## Sections

1. **Home** - Main landing with title and animated elements
2. **About** - Information about ALvVERSE
3. **FAQ** - Frequently asked questions
